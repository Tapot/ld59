# LD59

Repository root for the LD59 game jam project.

## Structure

```text
ld59/
├── ld59_game/   # Godot project
├── .gitignore
└── README.md
```

## Getting Started

1. Open the `ld59_game` folder in Godot.
2. Use Godot to run the main scene or project.
3. Commit only source assets, scenes, scripts, and project configuration.

## Notes

- Temporary files, editor caches, import data, and operating system junk are excluded by `.gitignore`.
- Keep exported builds and local test artifacts outside the repository when possible.
